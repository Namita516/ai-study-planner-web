# AI Study Planner

This repository contains a responsive AI Study Planner website built with HTML, CSS, and JavaScript.

## Project structure

- `index.html` - page markup and hero section layout
- `style.css` - dark theme styling, animations, and responsive layout
- `script.js` - form handling, plan generation, and download functionality

## Features

- Modern dark theme UI
- Responsive hero section with title and plan form
- Inputs for name, available study hours, and goal selection
- Personalized timetable generation based on selected study goal
- Clean schedule cards for each study session
- Download button saves the plan as `study_plan.txt`
- Smooth fade and pop animations

## Usage

1. Open `index.html` in a browser.
2. Enter your name, available study hours, and select your study goal.
3. Click `Generate Study Plan` to create your timetable.
4. Use the `Download Study Plan` button to save the plan as a text file.

## Notes

- The generated timetable adapts dynamically to available hours.
- The download action saves a plain text plan for offline use.
